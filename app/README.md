# Cepheus Build GUI

Lightweight Flutter desktop console for Cepheus Build.

## Run

```bash
cd app
flutter run -d macos
```

## History

Local run history is stored in the toolkit repo at:

```text
history/build-history.json
```

Commit that file when the team should share run history across machines.

## GitHub Dispatch

GitHub mode uses `gh workflow run` against each product repo's
`shared-build.yml` workflow. Choose GitHub-provided or org self-hosted runners
from the runner segmented control; org self-hosted maps to the standard
`["self-hosted","linux|macos|windows"]` labels generated by Cepheus Build.
